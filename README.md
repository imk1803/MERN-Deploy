# MERN Stack E-commerce Project

## Tính năng

- Xác thực người dùng (Đăng ký, Đăng nhập, Đăng xuất)
- Phân quyền Admin/User
- Quản lý sản phẩm
- Tìm kiếm và lọc sản phẩm
- Giỏ hàng và Thanh toán
- Responsive design

## Công nghệ sử dụng

- MongoDB & Mongoose
- Express.js
- React.js & Redux Toolkit
- Node.js
- TailwindCSS
- JWT Authentication

## Cài đặt

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```
